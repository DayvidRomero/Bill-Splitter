## Bill Splitter

A Python command-line program that calculates and splits a 
restaurant bill between a party, including tax and tip.

### Features
- Split evenly by total check amount
- Split by individual item per person
- Automatically calculates tax and tip per person
- Displays itemized breakdown of subtotal, tax, tip, and total

### Concepts practiced
- User input handling and type conversion
- Conditional logic (if/elif/else)
- While loops
- Float arithmetic and rounding
- F-strings and formatted output
